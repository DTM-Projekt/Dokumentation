## Schnipsel
Diese Codebeispiele sollen später direkt in die Dokumentation übernommen werden.
```
>>> " ".join(['Eins','Zwei','Drei'])
'Eins Zwei Drei'

>>> "".join(map(lambda x:'  '+str(x),range(1,10)))
'  1  2  3  4  5  6  7  8  9'
```
